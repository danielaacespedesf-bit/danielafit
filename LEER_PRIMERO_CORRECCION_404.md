# DanielaFit v4 - corrección de imágenes 404

Esta versión corrige el error 404 de imágenes en iPhone.

Las imágenes ahora van embebidas dentro de app.js, por eso ya no dependen de que la carpeta exercises/ quede bien subida.

Para actualizar GitHub Pages:
1. Descomprime este ZIP.
2. Entra a la carpeta descomprimida.
3. Sube a GitHub los archivos de adentro, no la carpeta completa.
4. Reemplaza index.html, styles.css, app.js, manifest.json, service-worker.js e icons/.
5. Haz commit.

Si en iPhone sigue saliendo la versión vieja, borra los datos de Safari para tu sitio de GitHub Pages o elimina y vuelve a agregar la app a la pantalla de inicio.
