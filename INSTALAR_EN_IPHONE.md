# Cómo instalar DanielaFit en iPhone desde Windows

## Requisito

Necesitas que la app esté publicada en una URL HTTPS. La opción más simple es GitHub Pages.

## Pasos rápidos

1. Sube los archivos a GitHub Pages.
2. Abre la URL en Safari desde el iPhone.
3. Toca el botón Share/Compartir.
4. Toca Add to Home Screen / Agregar a pantalla de inicio.
5. Deja activado Open as Web App si aparece.
6. Toca Add / Agregar.
7. Abre DanielaFit desde el ícono del iPhone.

## Nota de privacidad

El código de la app puede estar público si usas un repositorio público de GitHub Pages. Pero tus registros diarios, peso, dolor, energía, cargas y progreso se guardan localmente en el iPhone y no se suben a GitHub.

## Para probar en tu laptop Windows

Puedes abrir `index.html` directamente en Chrome/Edge. Para probarla como PWA/offline en iPhone, usa una URL HTTPS como GitHub Pages.
