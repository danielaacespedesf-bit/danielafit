# DanielaFit v7 - Visuales sin imágenes externas

Esta versión elimina por completo el sistema anterior de imágenes externas.

No hay rutas a `exercises/*.svg`, no hay `<img src=...>` y no depende de archivos de imagen.
Cada visual se dibuja directamente dentro de la tarjeta del ejercicio con HTML/SVG generado por la app.

Para probar en iPhone abre:

`index.html?v=7.0.0`

Luego entra a Perfil y confirma que diga: `v7.0 visual-cards`.
